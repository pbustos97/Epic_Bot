# Epic Bot
### Bot that does commands per server
Prefix will be the same for all servers. It is defined when the bot runs from the terminal.

Dependencies
------
1. Python 3.5.6
2. [Discord.py 1.2.3]
3. [Numpy 1.16.4]

Default Commands
------
Found in [epic_bot_default_cmds.py]

[Numpy 1.16.4]:http://numpy.org
